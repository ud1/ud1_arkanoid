Управление:
Esc - Пауза
A - Наклонить платформу влево
D - Наклонить платформу вправо
R - Использовать один из резервных шаров

В игре есть следующие бонусы (появляются при уничтожении некоторых кирпичиков)
1. Дополнительный шар
2. Удлинение платформы, прирост очков сокращается на 50%
3. Укорачивание платформы, прирост очков увеличивается на 50%
4. Изменение гравитации влево, прирост очков увеличивается на 50%
5. Изменение гравитации вправо, прирост очков увеличивается на 50%
6. Ускорение течения времени, прирост очков увеличивается на 50%
7. Замедление течения времени, прирост очков сокращается на 50%

Также прирост очков зависит от скорости всех шаров.
